# Sistema de Autenticação (Front-End)

Este projeto consiste na implementação front-end de um fluxo completo de autenticação de usuários, incluindo login, cadastro, recuperação de senha e validação por código. Foi desenvolvido com o objetivo de praticar construção de interfaces, responsividade e lógica de interação no lado do cliente utilizando JavaScript.

## Funcionalidades

* Interface de login
* Fluxo de cadastro de usuários
* Processo de recuperação de senha
* Validação de código (simulação)
* Layout responsivo para diferentes dispositivos

## Tecnologias Utilizadas

* HTML
* CSS
* JavaScript

## Estrutura do Projeto

A aplicação foi organizada de forma a separar responsabilidades entre estrutura (HTML), estilização (CSS) e comportamento (JavaScript), facilitando a leitura e manutenção do código.

## Como Executar

1. Clone o repositório:
   git clone https://github.com/seu-usuario/seu-repositorio.git

2. Acesse a pasta do projeto

3. Abra o arquivo `index.html` em um navegador

## Objetivo

Simular um sistema real de autenticação do ponto de vista do front-end, com foco na experiência do usuário, responsividade e controle de fluxo de interação.

## Principais Aprendizados

* Manipulação do DOM com JavaScript
* Controle de fluxo de autenticação
* Estruturação de projetos front-end
* Técnicas de responsividade com CSS

## Limitações

Este projeto é uma simulação em front-end e não possui integração com back-end ou mecanismos reais de autenticação.
